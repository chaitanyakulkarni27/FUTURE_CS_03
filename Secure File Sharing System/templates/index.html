Security Features

  • AES-256 Encryption: Military-grade encryption algorithm
  • PBKDF2: Password-Based Key Derivation Function with 100,000 iterations
  • CBC Mode: Cipher Block Chaining with random IV for each file
  • Salt: Unique salt for each encryption operation
  • HTTPS Ready: Encrypted in-transit communication (when deployed)
